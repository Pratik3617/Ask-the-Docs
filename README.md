# Ask the Docs – Retrieval Augmented Generation (RAG)

## Overview
Ask the Docs is a production-style Retrieval-Augmented Generation (RAG) application that allows users to upload a document and ask questions grounded strictly in the document’s content.

## Scope
- Supported file formats: PDF (text-based), TXT
- Single-document ingestion
- CPU-only inference
- Deterministic responses
- No OCR
